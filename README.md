📊 Sales & Profit Analysis Dashboard (Power BI)

📌 Project Overview
This project analyzes retail sales data to evaluate business performance across regions, product categories, and customer segments. The goal is to identify trends, improve profitability, and support data-driven decision-making.

🎯 Objectives
- Analyze sales and profit performance  
- Identify loss-making products  
- Understand customer segments and top customers  

🛠 Tools & Technologies
- Power BI  
- Power Query  
- DAX  

📈 Key Features
- KPI cards: Total Sales, Total Profit, Profit Margin, Total Orders  
- Sales analysis by Region, Category, and Time  
- Profit analysis including loss-making products  
- Customer insights dashboard  
- Interactive slicers (Region, Category, Year)  

🧠 Key Insights
- Some products generate high sales but negative profit  
- Regional differences in profitability observed  
- A small group of customers contributes most revenue  

📂 Dataset
Superstore Sales Dataset (public dataset)
